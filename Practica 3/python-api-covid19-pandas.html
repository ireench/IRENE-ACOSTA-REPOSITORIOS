Uso de la API del Covid19 con pandas¶

API como una especificación formal que se establece cómo un módulo de un software, es decir, se comunica o interactúa con otro para cumplir una o muchas funciones. En este práctica se hará uso de la librería de código abierto de Python, pandas, que es una librería de código abierto con la que podemos analizar datos desde diferentes tipos de archivos (html, json, csv...), los modemos manipular y cargar en nuestro notebook de Jupyter.

La URL que vamos a utilizar para importar nuestros datos es la siguiente: https://api.covid19api.com/countries

Crear Variable¶

Se asignan con el igual = y se escriben entre comillas ' ' porque es una cadena de caracteres, lo que se denomina string. Lo que hacemos en este apartado es básicamente definir la variable URL.

url = 'https://api.covid19api.com/countries'

Crear Dataframe¶

Se define DataFrame a través de df, se trata de una herramienta por la cual podemos podemos organizar y almacenar una estructura de datos. Del mismo modo con anterioridad hemos importado la biblioteca ya instalada pandasa través de la convección pd. En pandas hay una función que se denomina como pd.read_json(url) y lo que hace efectivamente es leer los datos que se encuentran dentro de un archivo json.

VS es como definimos este dataframe¶

En el antiguo caso se concatena lo que ya vemos, pero si queremos crearlo ponemos algo es igual a algo (=) y podemos poner después versus, comparación etc. Se puede poner lo que sea, en el caso del inicio el import pandas pd si que teníamos que ponerlo así, pero en este caso no es necesario. Si llamamos a esta selección o concatenación nos sale el resultado, pero aquí estamos a esta selección llamándola vs, por eso como se ve abajo no se cambia nada después, solamente lo hemos creado con un nombre. En conclusión, VS es como definimos este dataframe
